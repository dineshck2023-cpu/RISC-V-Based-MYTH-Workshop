Welcome to the exciting world of digital logic design! In this journey, we will learn TL-Verilog, a modern hardware design language developed by Redwood EDA. We will use Makerchip, an online platform where we can write code, simulate circuits, and view outputs easily.

Day 1: RISC-V ISA & GNU Compiler Toolchain

Subtopics: RISC-V Basic Keywords, RISC-V Software Toolchain, Integer Number Representation

On the first day, you learn the fundamentals of the RISC-V Instruction Set Architecture (ISA), including important terms and concepts used in processor design. You also explore the RISC-V software toolchain, which helps in writing, compiling, and executing programs. In addition, you study how integers are represented in binary form inside a computer system.

Day 2: ABI & Basic Verification Flow

<img width="1520" height="720" alt="WhatsApp Image 2026-06-02 at 8 31 00 PM (2)" src="https://github.com/user-attachments/assets/5dc83e17-1c16-4ac2-bebb-038f4fd2483f" />
Subtopics: ABI Basics, ABI Labs

The second day focuses on the Application Binary Interface (ABI), which defines how software communicates with hardware and how functions interact with each other. Through ABI lab exercises, you gain practical experience and learn basic verification techniques to ensure that programs and processor operations work correctly.

Day 3: Digital Logic with TL-Verilog in Makerchip IDE

<img width="1520" height="720" alt="WhatsApp Image 2026-06-03 at 10 45 49 AM (4)" src="https://github.com/user-attachments/assets/c02c7bda-48ad-4a3e-9628-51d1bba5d017" /> <img width="1520" height="720" alt="WhatsApp Image 2026-06-03 at 11 29 46 AM" src="https://github.com/user-attachments/assets/fe8ef83c-af7a-411d-9b04-d6d8df7183ea" />
Subtopics: Logic Gates, Makerchip Platform, Combinational Logic, Sequential Logic, Pipelined Logic, Validity

On this day, you begin learning digital hardware design using TL-Verilog in the Makerchip IDE. You start with logic gates, which form the foundation of digital circuits. The session then covers combinational logic, sequential logic, pipelined logic, and validity concepts, helping you understand how digital systems process and manage data efficiently.

Subtopics: Simple RISC-V Microarchitecture, Fetch & Decode, RISC-V Control Logic

The fourth day introduces the structure of a simple RISC-V processor. You learn about the processor microarchitecture and implement key stages such as instruction fetch and instruction decode. You also develop the control logic responsible for coordinating different processor operations and executing instructions correctly.

Subtopics: Understanding CPU Pipelining, Solutions to Pipeline Hazards, Completing RISC-V Design

On the final day, you study CPU pipelining, a technique that improves performance by processing multiple instructions simultaneously. You learn about common pipeline hazards and methods to resolve them. Finally, you combine all the modules developed throughout the workshop and complete the design and verification of a functional RISC-V processor.
